# my-codex-plugin

Better Codex plugin for Claude Code

## Usage

```shell
claude plugin marketplace add nekohasekai/my-codex-plugin
claude plugin install my-codex-plugin@my-codex-plugin
claude "/my-codex-plugin:setup-codex"                                                                                                                                "
```

### Force Update

```shell
claude plugin marketplace update my-codex-plugin
claude plugin remove my-codex-plugin@my-codex-plugin
claude plugin install my-codex-plugin@my-codex-plugin
```
